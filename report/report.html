Normals

Given a point p1 on a sphere centered in a point p2, we can calculate the normals using the following formula:

n = normalize(p1 - p2)

UVs

Given a point p1 on a sphere with radius r we can calculate the UVs as follow:

  • u = atan2(p1.z, p1.x);

  • v = acos(p1.y / r);
